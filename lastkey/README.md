# lastkey

A confidential smart contract project built on [Zama FHEVM](https://docs.zama.ai/fhevm)
using the `blank` template. Generated by [fhevm-forge](https://github.com/yourusername/fhevm-forge).

## Quick Start

```bash
# Install dependencies
forge install

# Run tests (local FHE mock — no Gateway needed)
forge test

# Check for FHEVM-specific bugs
fhevm-forge lint ./src

# Estimate FHE gas costs
fhevm-forge gas

# Deploy to Sepolia testnet
cp .env.example .env
# Fill in SEPOLIA_RPC_URL and DEPLOYER_PRIVATE_KEY
fhevm-forge deploy --chains sepolia --contract ConfidentialVault
```

## Project Structure

```
src/          Solidity contracts (TFHE encrypted types)
test/         Forge tests (forge-fhevm local mock)
script/       Deployment scripts
lib/fhevm/    TypeScript SDK helpers
lib/hooks/    React hooks for FHEVM
agent/        Headless agent runtime
```
